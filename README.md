# Urkit

Urkit is a native C++ runtime modding framework for Unity games using Mono or IL2CPP.

It provides game-specific SDK generation for developing native C++ mods and includes a runtime loader for executing compatible mod DLLs.

By default, mods are loaded from the `mods` directory. The mod directory can be changed through the configuration.

## Features

* Mono and IL2CPP support
* Native C++ mod development
* Runtime mod loading
* Configurable mod directory
* Built-in runtime helper APIs
